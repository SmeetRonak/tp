# Project Portfolio: Rehaan Mahmood

## Project: CCAManager
CCAManager is a CLI-based application designed for Hall Leaders to manage Resident CCA records, track event participation, and analyze performance points efficiently.

---

## Summary of Contributions

### 1. Viewing Residents
* Implemented the `view-resident` command to allow the user to view a list of all the residents in the hall.
* Displays the residents in a list with their matriculation numbers.

### 2. CCA Statistics
* Implemented the `cca-stats` command to display statistics related to CCAs.
* Displays a list of all CCAs with their average points across all registered residents, and displays the most active resident for each CCA.
* Displays the most popular CCAs which are computed as the CCAs with the highest average points.

### 3. Resident Statistics
* Implemented the `resident-stats` command to display statistics related to residents.
* Displays a list of all residents with their total points across all their CCAs.
* Displays the most active residents in the hall based on their total points.

### 4. Help Command
* Implemented the `help` command to display a list of all available commands and their usage.

### 5. Tesing
* Wrote unit tests for `cca-stats` and `resident-stats`.
* Wrote automated text-UI test.
